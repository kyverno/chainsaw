# add

## Signature

`add(any, any)`

## Description

Does arithmetic addition of two specified values of numbers, quantities, and durations.

## Examples

### With numbers

```
add(`1`, `2`) == `3`
```

### With durations

```
add('1h', '2h') == '3h'
```

### With quantities

```
add('1Mi', '2Mi') == '3Mi'
```
