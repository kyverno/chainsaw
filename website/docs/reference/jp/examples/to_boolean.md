# to_boolean

## Signature

`to_boolean(string)`

## Description

It returns true or false for any string, such as 'True', 'TruE', 'False', 'FAlse', 'faLSE', etc.

## Examples

```
to_boolean('true') == `true`
```

```
to_boolean('True') == `true`
```

```
to_boolean('TRUE') == `true`
```

```
to_boolean('false') == `false`
```

```
to_boolean('False') == `false`
```

```
to_boolean('FALSE') == `false`
```
